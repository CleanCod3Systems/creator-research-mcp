# @cleancod3/providers

Internal dependency of [`creator-research-mcp`](https://www.npmjs.com/package/creator-research-mcp) —
one content-fetching adapter per platform (YouTube, TikTok, Instagram, Twitter/X, LinkedIn, web
articles, PDF, local files) for the Creator Research MCP server. Not meant to be installed on
its own.

Source and docs: [github.com/CleanCod3Systems/creator-research-mcp](https://github.com/CleanCod3Systems/creator-research-mcp).
